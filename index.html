Linked Edit Rules

Linked Edit Rules (LER) is a methodology to publish, link, combine and execute edit rules on the Web as Linked Data to verify consistency of Linked Statistical Data.

Learn more »

Edit Rules in RDF

Edit rules (or edits) are rules that encode a huge variety of statistical data consistency constraints. For example, in a statistical dataset with demographic information on a population, a man cannot be pregnant, an underage cannot possess a driving license, and a child can't be married. With LER, these constraints are encoded in Web standard formats and languages, such as RDF.

Web Distributed

In LER, edit rules are expressed as SPARQL, combining the RDF Data Cube model and Semantic Web rule models. With these, edit rules can be distributed on the Web (in a site like this one) and later retrieved and combined to customize an inference engine. A lightweight LER vocabulary models the rules, their scope and their statistical components.
